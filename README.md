# Proyecto-móvil

Con este proyecto los técnicos podrán sacar fotos de sus ticket y de los alimentos a la hora de comer, para ello lo primero que deben hacer es inicar sesión pulsando el botón arriba a la derecha 
<img width="493" height="826" alt="image" src="https://github.com/user-attachments/assets/2b0caa8b-1a3a-4bc6-9620-9d1e390ef3f7" />

Una vez pulsado el botón, deberán introducir su código de técnico.
<img width="492" height="824" alt="image" src="https://github.com/user-attachments/assets/ea7f4c6a-dfae-4073-a6ce-9876ad9dcbbd" />


Este dato se guardará en el localStorage para poder enviarlo automáticamente a la base de datos junto con el resto del formulario.
<img width="378" height="808" alt="image" src="https://github.com/user-attachments/assets/b314e966-538c-45b6-a289-b640ff46bc16" />

Y en el localstorage guardaremos su código y el nombre del mismo para introducirlo en la base de datos
<img width="853" height="276" alt="image" src="https://github.com/user-attachments/assets/08a54ae1-0288-4397-bb98-6d912e242eeb" />


Introduciremos todos los datos del formulario. además que si pulsamos sobre el icono de subir un archivo si nos encontramos en un dispositivo móvil se nos abrirá la camera para sacar una foto
<img width="378" height="806" alt="image" src="https://github.com/user-attachments/assets/73062eb5-32dc-4387-97ee-13d5391eceec" />

Como estamos en un pc se abrirá un filechooser
<img width="1180" height="663" alt="image" src="https://github.com/user-attachments/assets/356def8c-8c5c-4c2d-9fa5-fe848259cd23" />

Pudiendo previsualizar las fotos que hemos sacado:
<img width="354" height="501" alt="image" src="https://github.com/user-attachments/assets/06f4e686-0af9-426f-8d2c-1a1663200b16" />


Y simplemente pulsamos "Enviar" para introducirlos en la base de datos
